# ops-reduction

운영 부담 감소 자동화 3종. **소통·정보·리포트** — 매주 반복되는 시간 도둑 제거.

## 이 카테고리의 목적

마케터의 시간을 가장 많이 빼앗는 건 새로운 기획이 아니라 **반복 운영**이다. 댓글 답변, 경쟁사 체크, 주간 리포트 — 자동화하면 주 5~10시간 회수.

## 포함 자동화

| # | 자동화 | 회수 시간 (주당 추정) |
|---|---|---|
| [03](./03-unified-inbox/) | 댓글·DM·문의 통합 인박스 | 3~5시간 |
| [04](./04-competitor-intel/) | 경쟁사 인텔리전스 봇 | 2~3시간 |
| [09](./09-weekly-report/) | 주간 리포트 + 액션 | 3~4시간 |

## 데이터 흐름

```
[IG·YT·블로그·메일]
    │
    ▼
[03 통합 인박스] ─▶ Slack 채널 (분류·답변 초안) ─▶ 사람 1-click 발송

[경쟁사 도메인·SNS]
    │
    ▼
[04 경쟁사 인텔] ─▶ Slack 알림 (변경 사항만) ─▶ Notion 인사이트 DB

[모든 채널 KPI]
    │
    ▼
[09 주간 리포트] ─▶ 월요일 9시 Slack ─▶ 이번주 액션 3개 추천
```

## 의존성

3개 자동화 모두 **다른 자동화와 독립적으로 실행 가능**. 병렬 구축 OK.
단, **09는 다른 자동화의 실행 로그도 집계**하므로 가능하면 다른 것들이 어느 정도 자리 잡은 후.

## 구축 순서 (권장)

1. **03 먼저** — 가장 즉각적인 시간 절약 (답변 누락 = 매출 손실)
2. **04** — 정보 수집 자동화
3. **09** — 다른 자동화 결과까지 통합되도록 마지막
