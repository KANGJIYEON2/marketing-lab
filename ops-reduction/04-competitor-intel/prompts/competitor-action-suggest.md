# competitor-action-suggest

`our_action` 필드가 일반적인 가이드를 따르도록 하는 메타 프롬프트. **별도 호출이 아니라** `competitor-change-analyze.md`의 일부 출력 필드. 이 문서는 액션 품질의 기준을 정의.

## 좋은 액션의 4가지 조건

1. **시간 제약 명시** — "이번 주", "다음 분기", "다음 광고 시 테스트"
2. **단일 결정 또는 실행** — "X 또는 Y 중 결정" / "Z 추가"
3. **누가 / 어떤 자동화와 연결되는가** — 가능하면 다른 자동화 폴더 언급
4. **우리 차별점 보존** — 단순 모방이 아닌 차별화 또는 무대응 선택

## 인텐트별 액션 패턴

### `pricing` 변경
- 우리 가격대와 겹치면 → "이번 주 내 가격 검토 (이사회/혼자 결정)"
- 우리보다 비싸지면 → "우리 가성비 메시지 강화 (LP 헤드라인 A/B)" → 10번 자동화 연계
- 우리보다 싸지면 → "차별화 가치 강조 vs 매칭 중 결정"
- 첫 수집 → "informational only"

### `positioning` 변경
- 우리 핵심 메시지와 충돌 → "우리 메시지 재확인 + LP 카피 보강"
- 새 앵글 시도 → "관찰 후 효과 좋으면 다음 분기 검토"

### `feature` 변경
- 우리에게 없는 기능 → "로드맵 추가 검토 또는 의도적 미보유 결정"
- 우리도 있는 기능 → "기존 기능 마케팅 강화"

### `content` 변경 (블로그 새 글)
- SEO 경쟁 키워드 글 → "유사 토픽 글 작성, 우리 시각으로" → 02 콘텐츠 머신 연계
- 우리 잘 다루지 않은 주제 → "콘텐츠 캘린더 추가" → 01 아이디어 엔진에 시드

### `launch` (신제품·캠페인)
- 직접 경쟁 → "우리도 캠페인 검토" 또는 "더 강한 차별화로 대응"
- 인접 영역 → "사용자 반응 모니터링"

## 액션 안티패턴 (LLM이 빠지면 안 되는)

❌ "트렌드 모니터링" — 액션 아님, 측정 불가능
❌ "검토 필요" — 누가, 언제, 무엇을?
❌ "마케팅 강화" — 어떻게? 어느 채널?
❌ "고객에게 알림" — 어떤 메시지로, 어디서?
❌ "전략 회의 소집" — 1인 사업자에게는 무의미

## 액션 좋은 예 (이렇게 나와야)

✅ "이번 주 안에 가격표 검토 — Starter 매칭 vs 가치 차별화 강조 중 결정. 결정되면 [10 LP A/B](../../revenue-impact/10-lp-ab-test/)로 헤드라인 변형 실험."

✅ "다음 광고 변형 1개에 'X분 만에' 후킹 테스트. [06 광고 진화](../../revenue-impact/06-ad-evolution/)의 다음 주 사이클에 추가."

✅ "유사 비교 글 작성. 우리 차별점(셀프호스팅) 명시. [01 아이디어 엔진](../../content-machine/01-idea-engine/) ContentIdeas에 'n8n vs Make vs Us' 추가, 우선순위 high."

✅ "informational only — 가격대 다름, 우리 타겟과 무관."

## 시스템 메시지 보강 (v2 권장)

위 가이드를 `competitor-change-analyze.md`의 시스템 메시지에 few-shot으로 4~6개 예시 추가하면 LLM 액션 품질이 크게 올라감. MVP 운영하며 누적된 좋은 예시로 점진적 추가.

## 측정

- 매월 `our_action` 필드의 실제 실행률 추적
- 실행률 < 30% → 액션이 너무 추상적이거나 부적합. 프롬프트 보강
- 실행률 > 70% → 가이드 작동. 신뢰도 ↑
